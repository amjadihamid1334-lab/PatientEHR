# PatientEHR

Healthcare communication platform for doctors to manage patient appointments and communications.

## About

PatientEHR is a React Native mobile application that enables healthcare providers to:
- Send SMS appointment reminders (Twilio)
- Conduct video consultations
- Manage patient communications
- Use AI-powered medical documentation (Claude API)
