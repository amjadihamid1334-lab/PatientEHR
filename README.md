# PatientEHR - Healthcare Communication Platform

A React Native mobile application designed for communication between doctors and patients.

## Overview

PatientEHR is a mobile APP platform that enables healthcare providers to manage patient appointments, send notifications, conduct video consultations, maintain efficient communication workflows and chart the patient's visit.

## Features

### Communication Services
- **SMS Notifications**: Automated appointment reminders and health alerts via Twilio
- **Email Integration**: Professional email communications through SendGrid
- **Video Calling**: Secure telemedicine consultations with real-time communication
- **Multi-Doctor Support**: Each physician can configure their own communication services

### Smart Features
- **AI-Powered Documentation**: Intelligent medical text prediction using Claude API
- **Context-Aware Suggestions**: Dynamic auto-completion based on medical specialty and patient context
- **Device-Specific UI**: Conditional feature display based on device capabilities

## Technology Stack

- **Frontend**: React Native (iOS & Android)
- **Navigation**: React Navigation
- **Communications**: 
  - Twilio (SMS & Voice)
  - SendGrid (Email)
  - Custom video calling integration
- **AI/ML**: Claude API for intelligent text prediction
- **Security**: Device Keychain for secure credential storage

## Compliance

- A2P 10DLC registered for SMS communications
- HIPAA-aware design practices
- Secure data handling and storage
